## How to use

1. Download code in your local device
```
git clone https://github.com/sandeepnewa10/techchhen
```

2. Install all the dependencies
```
  npm install
```

3. Starts the frontend development server on your local machine

```
  npm start
```

## Build for production

```
npm run build
```

4. view 

```
  http://localhost:3000/
```
